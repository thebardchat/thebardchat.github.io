The Hardware Stack

Everything runs local. No cloud subscriptions. No vendor lock-in. A Pi 5 with a RAID array, four Ollama nodes, and Tailscale mesh tying it together.

// Compute
  • Raspberry Pi 5 — 16 GB RAM, controller node
  • Pironman 5-MAX — NVMe RAID chassis by Sunfounder
  • RAID 1 — 2× WD Blue SN5000 2TB NVMe via mdadm
  • 8TB External — Seagate ANGEL_CLOUD (NTFS)
  • 4-Node Cluster — Pi 5 + Pulsar00100 + Bullfrog + Jaxton
// AI Stack
  • Ollama 0.17.7 — local LLM inference, port 11434
  • Cluster Proxy — 4-node routing, port 11435
  • shanebrain-3b — custom model (llama3.2:3b base)
  • nomic-embed-text — 768-dim vector embeddings
  • Weaviate 1.36.2 — 17 collections, text2vec-ollama
// Services
  • MCP Server v2.2 — 42 tools, port 8100, Docker
  • Mega Dashboard — port 8300, 20+ live panels
  • Buddy Claude — port 8008, Claude↔Gemini engine
  • Open WebUI — port 3000, Portainer port 9000
  • Angel Cloud Gateway — port 4200, FastAPI
  • ShaneBrain Alerter — 5 AM briefing, weekly report
// Network
  • Tailscale VPN — mesh across all 4 nodes
  • Tailscale Funnel — public HTTPS at ts.net URL
  • Caddy 2.11.2 — reverse proxy, ready for custom domain
  • theangel.com — IONOS, expires 2027
  • Pi LAN — 10.0.0.42 · Tailscale 100.67.120.6
